1.ls 查看当前目录
2.cd Desktop 进入桌面
3.mkdir 文件夹名 在桌面新建文件夹
4.cd 新建文件夹名 进入新建的文件夹里
5.touch 文件名 在文件夹里新建文件
6.git init 把这个目录变成git可以管理的仓库
7.git add . 表示添加所有的文件
8.git status 查看状态
9.git commit -m "注释" 提交
10.git branch 查看主线或分支
11.git branch 分支名 创建分支
12.git checkout 分支名 切换分支
13.git add . 保存分支的内容
14.git commit -m "注释" 提交分支的内容
15.git checkout master 切换到主线
16.git merge fenzhi 主线和分支融合
17.融合成功
18.git remote add origin git@github.com:goddess-miao/仓库名.git
19.git push -u origin master
20.如果修改完之后在添加下文件，然后保存，再输入git push origin master
